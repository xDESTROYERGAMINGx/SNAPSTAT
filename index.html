SnapStat version history Version history

Every release improves the survey workflow, from form generation to scanning, reviewing, and exporting results. What changed in each release.

0.1.0 FIRST RELEASE

Snapstat

Introduced the printable survey form, survey capture, automatic tallying, session history, individual response review, and report export via Word or Excel. Forms, scanning, review, and exports.

0.1.1 FIXING BUGS

Scanning Improvements

Improved row and column marker alignment, added calibration support, added auto-crop for better accuracy, added scan instructions, adjusted answer validation, improved the information tab, made scan error messages clearer for users, and fixed bugs. Better alignment, cropping, guidance, and fixes.

0.1.2 CURRENT RELEASE

New Features and Improvements

Added device requirements, added terms and conditions, added search and edit features in individual responses, improved the scanning animation, slightly reduced scanning time, adjusted the information/instruction box, locked the “ballpen or marker only” instruction from being changed, and fixed bugs. Faster scans, response editing, guidance, and fixes.

0.1.3 UPCOMING UPDATE

Upcoming Features

Added coach marks to guide new users and improved response scale configuration. Coach marks and better scale setup.
